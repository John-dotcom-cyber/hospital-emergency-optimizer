# 🚀 MVP — Hospital Emergency Optimizer

## 1. Objectif du MVP

Créer une première version fonctionnelle démontrant la valeur du produit, avec un périmètre réduit mais impressionnant.

---

## 2. Contenu du MVP

### 🔮 1. Prédiction simple du flux d’arrivées
- modèle Prophet ou SARIMA
- prédiction horaire ou journalière
- visualisation simple

### 🧪 2. Simulation basique
- file d’attente M/M/1
- paramètres modifiables :
  - taux d’arrivée
  - capacité
  - personnel

### 🧠 3. Optimisation minimale
- allocation simple du personnel
- contraintes basiques

### 📊 4. Dashboard Streamlit
- prédictions
- simulation
- recommandations simples

---

## 3. Données nécessaires
- historique des urgences (anonymisé)
- météo (API open)
- planning du personnel (exemple fictif)

---

## 4. Livrables du MVP
- notebook de prédiction
- notebook de simulation
- notebook d’optimisation
- dashboard Streamlit
- documentation
